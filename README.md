# 路亚鱼护

一款专为路亚钓鱼爱好者设计的鱼获记录工具，帮助钓友记录每次出钓的渔获。

## 功能特点

- 📷 **拍照记录** - 快速拍摄鱼获照片
- 📏 **尺寸记录** - 输入长度，自动估算重量
- 📍 **定位记录** - 自动记录钓获地点
- 🏷️ **装备管理** - 管理鱼竿、鱼轮、鱼饵，支持设置默认装备
- 🔍 **筛选排序** - 按时间/尺寸/重量筛选和排序
- 📊 **统计分析** - 今日/本月/本年/全部渔获统计
- 🔄 **批量操作** - 支持多选删除
- 📤 **分享** - 一键分享渔获到社交平台

## 技术栈

- **Flutter** 3.11+
- **Riverpod** 状态管理
- **SQLite** 本地数据库
- **fl_chart** 图表统计
- **camera** 相机拍照
- **geolocator** 定位

## 项目结构

```
lib/
├── core/
│   ├── providers/     # Riverpod状态管理
│   └── services/      # 数据库服务
├── features/
│   ├── camera/        # 拍照记录页
│   ├── fish_detail/   # 鱼获详情页
│   ├── fish_list/     # 鱼护列表页
│   ├── home/          # 首页
│   ├── settings/      # 设置页
│   └── stats/         # 统计页
├── widgets/           # 通用组件
└── main.dart          # 入口
```

## 运行

```bash
flutter pub get
flutter run
```

## 数据

- 鱼获记录：品种、尺寸、重量、去向（放流/保留）、时间、地点
- 装备管理：鱼竿、鱼轮、鱼饵（品牌、型号、专业参数）
- 品种历史：自动记录常用品种
